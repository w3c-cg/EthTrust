# EthTrust Community Group

This repository is being used for work in the W3C EthTrust Community Group, governed by the [W3C Community License
Agreement (CLA)](http://www.w3.org/community/about/process/cla/). To make substantive contributions,
you must join the CG.

If you are not the sole contributor to a contribution (pull request), please identify all
contributors in the pull request comment.

To add a contributor (other than yourself, that's automatic):

### Using the command line: 

mark them one per line as follows:

```
+@github_username
```

If you added a contributor by mistake, you can remove them in a comment with:

```
-@github_username
```

If you are making a pull request on behalf of someone else but you had no part in designing the
feature, you can remove yourself with the above syntax.

### Using the Web Interface

If you are using github.com to create or commit the Pull Request, you can add a contributor by adding, in a comment

```
Co-authored-by: Contributor Name <email@addre.ss>
```
